## 4. AUROC internal 圖修圖過程與程式碼

修圖流程另存成可開啟的 PowerShell 檔：[0603_fix_AUROC_internal_layout.ps1](./0603_fix_AUROC_internal_layout.ps1)。下方保留逐步修正過程，包含檢查失敗的中間版與最後採用的正式版。

### 4.1 第一次檢查：只加外框白邊

第一次處理先將圖四周加白邊，避免插入 Word/PDF 時外緣被裁切。但檢查後發現 Y 軸標題仍與刻度文字太近，因此此版不作為最後版。

```powershell
$src = 'c:\GitHub\AI4JUBO\Revision\20260603\0603_Figure1_ROC_internal_cv_revised_1to1.png'
$backup = 'c:\GitHub\AI4JUBO\Revision\20260603\0603_Figure1_ROC_internal_cv_revised_1to1_original.png'

Add-Type -AssemblyName System.Drawing
if (-not (Test-Path -LiteralPath $backup)) {
  Copy-Item -LiteralPath $src -Destination $backup
}

$img = [System.Drawing.Image]::FromFile($backup)
$left = 90; $top = 70; $right = 100; $bottom = 100
$bmp = New-Object System.Drawing.Bitmap ($img.Width + $left + $right), ($img.Height + $top + $bottom)
$g = [System.Drawing.Graphics]::FromImage($bmp)
$g.Clear([System.Drawing.Color]::White)
$g.DrawImage($img, $left, $top, $img.Width, $img.Height)
$g.Dispose()
$img.Dispose()
$bmp.Save($src, [System.Drawing.Imaging.ImageFormat]::Png)
$bmp.Dispose()
```

### 4.2 第二次檢查：重畫 Y 軸標題與刻度

第二次改用原始圖作為底圖，裁掉原本太靠左且會重疊的 Y 軸標題與刻度，再重新畫出 Y 軸刻度與標題。檢查後 Y 軸問題已解決，但圖例第一列文字仍被框線裁到。

```powershell
$src = 'c:\GitHub\AI4JUBO\Revision\20260603\0603_Figure1_ROC_internal_cv_revised_1to1_original.png'
$out = 'c:\GitHub\AI4JUBO\Revision\20260603\0603_Figure1_ROC_internal_cv_revised_1to1.png'

Add-Type -AssemblyName System.Drawing

  $rect = New-Object System.Drawing.RectangleF 112, ($topPad + $ys[$i] - 22), 82, 44

$labelFormat = New-Object System.Drawing.StringFormat
$labelFormat.Alignment = [System.Drawing.StringAlignment]::Center
```

## 建議回覆文字

1. We added the missing P values for the institutional region and institution-scale distributions in the included-versus-excluded comparison. The institutional region distribution differed between groups (P < 0.001), whereas the institution-scale distribution did not differ materially (P = 0.926).
2. We revised the calibration plots to include predicted-probability histograms while aligning the axis labels, tick presentation, and legend order with the revised manuscript figures.
3. Because Table 3 has been updated, we replaced the ROC curve and confusion-matrix figures with versions based on the final updated model-performance results. The internal AUROC figure was additionally adjusted to prevent the Y-axis label/ticks and legend text from being clipped.
